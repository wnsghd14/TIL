#### HTML

``` 
<!doctype html>:Html을 알리는 태그
<html (lang="ko")>:시작과 끝
<head>:브라우저에게 정보를 주는 태그
<body>: 몸통과 같은 존재(내용) 
<meta>: <meta charset="utf-8">
html5에서 한국어뿐만아니라 모든언어를 사용할수있는 문자세트
<div> 전체를 감싸는 요소
<title><br>:줄바꿈
<img>,<video>,<audio><header>: 머리말/
<nav>: 문서 +사이트 연결 내비게이션 역할
<ul>,<ol>,<li>: 목록
<p>:텍스트
<hn>:크기별 제목
<article>:콘텐츠
<aside>:사이드바
<footer>: 저작권표시
<table>,<tr>,<td>,<th>:표*
<form>,<fieldset>,<legnd>,<label>,<button>,<input> 등: 폼만들기

```
